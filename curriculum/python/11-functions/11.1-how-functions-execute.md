# Functions kaise execute hote hain?

```python
print  "Navgurukul"      	#Yahan pe Navgurukul print hoga

def say_hello():		# Yahan pe humne ek function likkha hahi say_hello name ka
    print "Hello!"      	# Yahan pe Hello print hoga
    print "Aap kesay ho?"    	# yahan pe Aap kesay ho print hoga?

say_hello() 			# Humne yahan  pe function ko call kia hai
print “Python is awesome”	# Yahan pe Python is awesome print hoga
say_hello() 			# Humne yahan  pe function ko dabara call kia hai
print "Hello"			# dobara print Hello kia humne 
say_hello()			# Humne yahan  pe function ko fir se call kia hai

```
